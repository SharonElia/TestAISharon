# Sharon Test Project

## Description
This is a simple Python project demonstrating basic functionality with unit tests.

## Files

### hello.py
A simple Python script that:
- Prints "Sharon Test, world!"
- Calculates the sum of 3 + 4
- Prints the result (7)

### test_hello.py
Unit tests for the hello.py script using Python's unittest framework.

**Tests include:**
- `test_calculation()` - Verifies that 3 + 4 equals 7
- `test_print_output()` - Verifies that the print statement outputs the correct string

## Running the Code

### Run the main script:
```bash
python hello.py
```

### Run the tests:
```bash
python test_hello.py
```
